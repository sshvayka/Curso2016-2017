Dataset Requirements

Our dataset is based at the City Of Calgary, Canada. You can check things like census, permits, so we think that this information fulfill the requirement "Data are in the smart cities domain".

Most data is avaliable at CSV format, so it fulfill the requeriment "Data are avaliable as a CSV file".

Under the Open Data Terms of Use we have "world-wide license to use, modify, and distribute the Data contained on this Site for any lawful Use", so it fulfill the requeriment "Data have an open license so it can be published".

You can check data related to places like parks, monuments, so we think that this information fulfill the requeriment "Data can be easily linked with generic real world entities".

The webpage provides a small description about the content of the dataset and the last update of it, so we think that this information fulfill the requeriment "Documentation exists for the data"
