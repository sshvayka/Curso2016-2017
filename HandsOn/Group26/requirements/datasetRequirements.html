Datasets requirements satisfiability chosen city: San Francisco In this document we will discuss up to what point does the datasets used satisfy the project dataset requirement: 1)Data are in the smart cities domain. According to Forbes magazine San Francisco ranks 4th in the "The World's 'Smartest' Cities" ranking. Additionally all the data contained in the CSV files provide useful information about cultural spots around the city. 2) Data are available as a CSV file. Every dataset used is an original CSV file 3) Data have an open license so it can be published. " 4)Data can be easily linked with generic real-world entities (e.g., locations). The data provided is developers-orientated "Explore, view, and download our data. Developers - check out our developer page for tips on API access and use." the main purpose of these datasets is to be able to link the information or data provided with the streets and maps of San Francisco. 5)Documentation exists for the data (optional).
