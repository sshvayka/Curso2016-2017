Dataset Requeriments

Los conjuntos de datos seleccionados deben cumplir con los siguientes requisitos:

  • Deben estar en el dominio de las ciudades inteligentes: El portal elegido para la obtencion de los datasets es http://datosabiertos.sevilla.org/ que esta destinado a la publicacion de datos abiertos referentes a Sevilla.
  • Deben estar en formato CSV: Todos los archivos estan en formato CSV.
  • Deben tener licencia abierta para poder ser publicados: Cuentan con una licenca CC 4.0
  • Deben enlazarse facilmente con entidades del mundo real: Se enlazan de forma sencilla con la red de transporte de Sevilla.
  • Existe documentación sobre los datos(Opcional): En la descripcion de cada dataset se indica la nomenclatura que se ha seguido para nombrar cada campo y que significa cada uno.
  • Pueden venir de múltiples fuentes(Opcional): Provienen del ayuntamiento de Sevilla y de empresas que tienen relacion con el transporte.
